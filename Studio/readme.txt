Convertigo Studio
=================

In order to compile Convertigo Studio project, you must import the whole project from
our SVN repository inside your Eclipse workspace.

Use the Classic eclipse distribution.

The following Eclipse plugins should have been installed:

* org.eclipse.jface.text
* org.eclipse.wst.xml.ui
* org.eclipse.wst.sse.ui
* org.eclipse.wst.sse.core
* org.eclipse.wst.jsdt.ui
* org.eclipse.wst.jsdt.core

Optionaly, you can consider installing Source control plugins (such as SVNkit).
